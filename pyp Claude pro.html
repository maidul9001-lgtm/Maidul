Recursive Function
▶

ফাংশন নিজেকে নিজে কল করে। Factorial বের করার উদাহরণ:

def factorial(n):
    if n == 1:
        return 1
    return n * factorial(n - 1)

print(factorial(5))  # 120 (5×4×3×2×1)

Class ও Object কী? গুরুত্বপূর্ণ
▶

Class হলো ছাঁচ/ডিজাইন, Object হলো সেই ছাঁচ থেকে তৈরি জিনিস। যেমন "গাড়ির ডিজাইন" হলো class, আর "আপনার গাড়ি" হলো object।

class Student:
    def __init__(self, naam, roll):
        self.naam = naam
        self.roll = roll

    def info(self):
        print(f"{self.naam} - রোল: {self.roll}")

s1 = Student("Maidul", 1)  # Object তৈরি
s1.info()  # Maidul - রোল: 1
Inheritance (উত্তরাধিকার) গুরুত্বপূর্ণ
▶

একটি class আরেকটির সব গুণ নিয়ে নতুন কিছু যোগ করতে পারে।

class Animal:
    def shwas(self):
        print("শ্বাস নিচ্ছি")

class Dog(Animal):  # Animal থেকে নেওয়া
    def bark(self):
        print("ঘেউ ঘেউ!")

d = Dog()
d.shwas()  # শ্বাস নিচ্ছি (Animal থেকে)
d.bark()   # ঘেউ ঘেউ! (Dog-এর নিজের)

    x = int(input("সংখ্যা দিন: "))

Decorator ফাংশন অ্যাডভান্সড
▶

একটি ফাংশনকে মোড়ক দিয়ে অতিরিক্ত কাজ করানো।

def log_karo(func):
    def wrapper(*args):
        print(f"কল হচ্ছে: {func.__name__}")
        return func(*args)
    return wrapper

@log_karo
def jog(a, b):
    return a + b

jog(3, 4)
# আউটপুট: কল হচ্ছে: jog → 7
Generator ও yield অ্যাডভান্সড
▶

Generator একবারে সব না দিয়ে একটা একটা করে মান দেয় — মেমোরি সাশ্রয়ী।

def count_karo(n):
    i = 0
    while i < n:
        yield i   # একটা দিয়ে থামে
        i += 1

for num in count_karo(5):
    print(num)  # 0 1 2 3 4
Regular Expression — re মডিউল অ্যাডভান্সড
▶

টেক্সটের মধ্যে প্যাটার্ন খোঁজার শক্তিশালী টুল।

import re

text = "আমার নম্বর 01711-123456"
phone = re.findall(r'\d+', text)
print(phone)  # ['01711', '123456']

email = "test@gmail.com"
if re.match(r'\S+@\S+\.\S+', email):
    print("বৈধ ইমেইল")
