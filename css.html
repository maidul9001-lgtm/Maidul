CSS পরিচিতিCSS কী ও কীভাবে কাজ করে
১-২ দিন ▼
CSS কী এবং কেন দরকার?
▶

CSS (Cascading Style Sheets) দিয়ে HTML-কে সুন্দর করা হয়। HTML হলো কাঠামো, CSS হলো সেই কাঠামোর রঙ, আকার ও সৌন্দর্য।

CSS ছাড়া ওয়েবসাইট দেখতে সাদা-কালো নোটপ্যাডের মতো। CSS দিয়েই প্রফেশনাল ডিজাইন তৈরি হয়।

CSS যুক্ত করার ৩ উপায় গুরুত্বপূর্ণ
▶ 
/* ১. বাইরের ফাইল — সেরা পদ্ধতি */
/* HTML-এ লিখুন: */
<link rel="stylesheet" href="style.css">

/* ২. head-এর ভেতরে */
<style>
  body { background: #fff; }
</style>

/* ৩. সরাসরি ট্যাগে — এড়িয়ে চলুন */
<p style="color: red;">লাল লেখা</p>
CSS সিনট্যাক্স ও কমেন্ট
▶

Selector → কোন এলিমেন্টে দেবো | Property → কী পরিবর্তন করবো | Value → কতটুকু বা কী রঙ

/* এটি CSS কমেন্ট */

p {                    /* selector */
  color: red;          /* property: value */
  font-size: 16px;
  font-weight: bold;
}
২
সিলেক্টর (Selectors)কোন এলিমেন্টে স্টাইল দেবো
৩-৪ দিন ▼
Element, Class ও ID সিলেক্টর গুরুত্বপূর্ণ
▶ 
/* Element — সব p ট্যাগে */
p { color: blue; }

/* Class — class="card" যেখানে আছে */
.card { background: #fff; padding: 20px; }

/* ID — id="header" যেখানে আছে */
#header { background: #1a1a2e; }

/* একসাথে একাধিক */
h1, h2, h3 { font-weight: bold; }

/* সব এলিমেন্ট */
* { box-sizing: border-box; }
Pseudo-class: hover, focus, nth-child
▶ 
/* মাউস রাখলে */
button:hover { background: #3b82f6; }

/* ইনপুটে ক্লিক করলে */
input:focus { border-color: blue; }

/* প্রথম এলিমেন্ট */
li:first-child { color: red; }

/* শেষ এলিমেন্ট */
li:last-child { color: green; }

/* জোড় সংখ্যার এলিমেন্ট */
tr:nth-child(even) { background: #f0f0f0; }
Pseudo-element: before, after
▶ 
/* এলিমেন্টের আগে কিছু যোগ */
h2::before {
  content: "★ ";
  color: gold;
}

/* এলিমেন্টের পরে কিছু যোগ */
a::after {
  content: " →";
}

/* প্রথম অক্ষর বড় করা */
p::first-letter {
  font-size: 2em;
  float: left;
}

  background-position: center;
  background-repeat: no-repeat;

  background: linear-gradient(135deg, #f97316, #ef4444);

  /* বৃত্তাকার */
  background: radial-gradient(circle, #3b82f6, #1a1a2e);

/* টেক্সটে গ্রেডিয়েন্ট */
h1 {

  -webkit-background-clip: text;
  -webkit-text-fill-color: transparent;
}

টেক্সট ও ফন্টfont, text-align, spacing
৩-৪ দিন ▼
ফন্ট স্টাইল করা গুরুত্বপূর্ণ
▶ 
p {
  font-size: 16px;        /* সাইজ */
  font-weight: bold;      /* মোটা (100-900) */

  font-family: 'Arial', sans-serif;

}

/* Google Fonts ব্যবহার */
@import url('https://fonts.googleapis.com/css2?family=Roboto&display=swap');
body { font-family: 'Roboto', sans-serif; }

}

/* লিংকের আন্ডারলাইন সরানো */

বক্স মডেল (Box Model)margin, padding, border, width
১ সপ্তাহ ▼
Box Model কী? গুরুত্বপূর্ণ
▶

প্রতিটি HTML এলিমেন্ট একটি বাক্সের মতো। বাইরে থেকে ভেতরে: margin → border → padding → content

.box {
  /* Content এর সাইজ */
  width: 300px;
  height: 200px;

  /* ভেতরের ফাঁকা জায়গা */
  padding: 20px;

  /* বাইরের ফাঁকা জায়গা */
  margin: 10px;

  /* বর্ডার */
  border: 2px solid #3b82f6;
}
Padding ও Margin বিস্তারিত
▶ 
.box {
  /* চারদিকে সমান */
  padding: 20px;

  /* উপর-নিচ, বাম-ডান */
  padding: 10px 20px;

  /* উপর, ডান, নিচ, বাম */
  padding: 5px 10px 15px 20px;

  /* আলাদাভাবে */
  padding-top: 10px;
  padding-right: 20px;
  padding-bottom: 10px;
  padding-left: 20px;

  /* মাঝে রাখতে */
  margin: 0 auto;
}
Border ও Border-radius
▶ 
.card {
  /* বর্ডার: মোটা, ধরন, রঙ */
  border: 2px solid #3b82f6;
  border: 1px dashed #ccc;   /* ড্যাশ */
  border: 3px dotted red;    /* ডট */

  /* কোণ গোল করা */
  border-radius: 8px;        /* চারকোণ */
  border-radius: 50%;        /* পুরো গোল */
  border-radius: 10px 0 10px 0; /* আলাদা */

  /* ছায়া */
  box-shadow: 0 4px 20px rgba(0,0,0,0.3);
}
width, height, max-width
▶ 
.container {
  width: 100%;       /* পুরো প্রস্থ */
  max-width: 1200px; /* সর্বোচ্চ প্রস্থ */
  margin: 0 auto;   /* মাঝে রাখা */

  /* উচ্চতা */
  height: 100px;
  min-height: 100vh; /* স্ক্রিনের পুরো উচ্চতা */
}
৬
Flexbox লেআউটআধুনিক লেআউটের সেরা পদ্ধতি
১ সপ্তাহ ▼
Flexbox শুরু করা গুরুত্বপূর্ণ
▶

Flexbox দিয়ে এলিমেন্ট পাশাপাশি বা উপর-নিচে সহজে সাজানো যায়।

.container {
  display: flex;

  /* দিকনির্দেশনা */
  flex-direction: row;     /* পাশাপাশি (default) */
  flex-direction: column;  /* উপর-নিচ */

  /* মোড়ানো */
  flex-wrap: wrap;  /* জায়গা না হলে নিচে যাবে */
}
justify-content ও align-items গুরুত্বপূর্ণ
▶ 
.container {
  display: flex;

  /* অনুভূমিক সাজানো */
  justify-content: flex-start;   /* বাম */
  justify-content: flex-end;     /* ডান */
  justify-content: center;       /* মাঝে */
  justify-content: space-between; /* দুই মাথায় */
  justify-content: space-around;  /* চারদিকে ফাঁক */

  /* উল্লম্ব সাজানো */
  align-items: center;    /* মাঝে */
  align-items: flex-start; /* উপরে */
  align-items: stretch;    /* টেনে লম্বা */
}
Flexbox দিয়ে পূর্ণ সেন্টার করা
▶

পেজের মাঝে কিছু রাখতে সবচেয়ে সহজ উপায়:

.center-box {
  display: flex;
  justify-content: center;
  align-items: center;
  min-height: 100vh;
}

/* নেভবার উদাহরণ */
nav {
  display: flex;
  justify-content: space-between;
  align-items: center;
  padding: 16px 24px;
}
flex: 1 ও gap
▶ 
.container {
  display: flex;
  gap: 20px;  /* আইটেমের মাঝে ফাঁক */
}

/* সমান জায়গা নেওয়া */
.item { flex: 1; }

/* ৩ কলামের কার্ড লেআউট */
.cards {
  display: flex;
  flex-wrap: wrap;
  gap: 16px;
}
.card {
  flex: 1 1 300px; /* min 300px, নয়তো বাড়বে */
}
৭
CSS Grid লেআউটদুই দিকে সাজানোর শক্তিশালী উপায়
১ সপ্তাহ ▼
Grid শুরু করা গুরুত্বপূর্ণ
▶

Grid দিয়ে সারি ও কলামে এলিমেন্ট সাজানো যায় — পত্রিকার লেআউটের মতো।

.grid {
  display: grid;

  /* ৩টি সমান কলাম */
  grid-template-columns: 1fr 1fr 1fr;

  /* shorthand */
  grid-template-columns: repeat(3, 1fr);

  /* গ্যাপ */
  gap: 20px;
}

/* রেসপন্সিভ কার্ড গ্রিড */
.cards {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));
  gap: 16px;
}
grid-column ও grid-row দিয়ে ঘর জুড়া
▶ 
.grid {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 16px;
}

/* ২টি কলাম জুড়ে দেওয়া */
.big-item {
  grid-column: span 2;
}

/* ২টি সারি জুড়ে দেওয়া */
.tall-item {
  grid-row: span 2;
}
৮
Responsive Designমোবাইল ও ডেস্কটপে সুন্দর দেখানো
১ সপ্তাহ ▼
Media Query গুরুত্বপূর্ণ
▶

স্ক্রিনের সাইজ অনুযায়ী আলাদা স্টাইল দেওয়া যায়।

/* মোবাইল ফার্স্ট (ছোট থেকে বড়) */

/* সব স্ক্রিনে */
.container { padding: 16px; }

/* ট্যাবলেট (768px বা বড়) */
@media (min-width: 768px) {
  .container { padding: 24px; }
  .grid { grid-template-columns: repeat(2, 1fr); }
}

/* ডেস্কটপ (1024px বা বড়) */
@media (min-width: 1024px) {
  .container { max-width: 1200px; margin: 0 auto; }
  .grid { grid-template-columns: repeat(3, 1fr); }
}
Responsive Units: %, vw, vh, rem, em
▶ 
.box {
  width: 50%;    /* parent-এর ৫০% */
  width: 100vw;  /* স্ক্রিনের পুরো প্রস্থ */
  height: 100vh; /* স্ক্রিনের পুরো উচ্চতা */

  /* rem = root font-size (সাধারণত 16px) */
  font-size: 1rem;   /* = 16px */
  font-size: 1.5rem; /* = 24px */

  /* em = parent font-size */
  padding: 1em;
}
Responsive Image ও Video
▶ 
/* ছবি কখনো overflow করবে না */
img {
  max-width: 100%;
  height: auto;
}

/* ভিডিও responsive */
.video-wrap {
  position: relative;
  padding-bottom: 56.25%; /* 16:9 ratio */
}
.video-wrap iframe {
  position: absolute;
  width: 100%;
  height: 100%;
}

অ্যানিমেশন ও অ্যাডভান্সডtransition, animation, variables
১ সপ্তাহ ▼
Transition (মসৃণ পরিবর্তন) গুরুত্বপূর্ণ
▶ 
button {
  background: #3b82f6;
  color: white;
  padding: 10px 20px;
  border: none;
  border-radius: 8px;

  /* মসৃণ পরিবর্তন: property, সময়, ধরন */
  transition: all 0.3s ease;
}

button:hover {
  background: #1d4ed8;
  transform: scale(1.05);  /* একটু বড় */
}
Transform: scale, rotate, translate
▶ 
.box {
  transform: scale(1.2);       /* ২০% বড় */
  transform: rotate(45deg);    /* ৪৫° ঘোরানো */
  transform: translateX(50px); /* ডানে সরানো */
  transform: translateY(-20px);/* উপরে সরানো */

  /* একসাথে */
  transform: scale(1.1) rotate(5deg);
}
Keyframe Animation অ্যাডভান্সড
▶ 
/* অ্যানিমেশন তৈরি */
@keyframes bounce {
  0%   { transform: translateY(0); }
  50%  { transform: translateY(-20px); }
  100% { transform: translateY(0); }
}

/* এলিমেন্টে লাগানো */
.ball {
  animation: bounce 1s ease infinite;
}

/* ফেড ইন */
@keyframes fadeIn {
  from { opacity: 0; }
  to   { opacity: 1; }
}
.hero { animation: fadeIn 1s ease; }
CSS Variables (Custom Properties) অ্যাডভান্সড
▶

পুরো সাইটে একটি রঙ পরিবর্তন করতে চাইলে CSS Variable দিয়ে এক জায়গায় পরিবর্তন করলেই হয়।

/* :root এ ভেরিয়েবল সংজ্ঞায়িত করুন */
:root {
  --primary: #3b82f6;
  --secondary: #8b5cf6;
  --bg: #0f0f1a;
  --text: #f1f5f9;
  --radius: 8px;
}

/* ব্যবহার */
button {
  background: var(--primary);
  border-radius: var(--radius);
  color: var(--text);
}
🎯 পরবর্তী ধাপ — JavaScript এর পথে

CSS শেষ হলে এই ক্রমে এগিয়ে যান:

১. Bootstrap / Tailwind CSS
   → রেডিমেড CSS ফ্রেমওয়ার্ক
   → দ্রুত সুন্দর UI বানানো যায়

   → বাটন ক্লিক, ফর্ম যাচাই
   → DOM manipulation

৩. প্রজেক্ট বানান
   → পোর্টফোলিও ওয়েবসাইট
   → ল্যান্ডিং পেজ
   → ব্লগ ডিজাইন

৪. এরপর React / Next.js
   → আধুনিক ওয়েব ডেভেলপমেন্ট
