HTML কী এবং কীভাবে কাজ করে?
▶

HTML (HyperText Markup Language) হলো ওয়েবপেজের কাঠামো তৈরির ভাষা। ব্রাউজার HTML পড়ে এবং সুন্দরভাবে দেখায়।

HTML → ওয়েবপেজের কাঠামো | CSS → সৌন্দর্য | JavaScript → কার্যকারিতা

VS Code সেটআপ ও প্রথম ফাইল তৈরি
▶

VS Code ডাউনলোড করুন → Live Server এক্সটেনশন ইনস্টল করুন → নতুন ফাইল তৈরি করুন index.html নামে।

মোবাইলে কাজ করতে Spck Code Editor বা Acode অ্যাপ ব্যবহার করুন।

প্রথম HTML পেজ তৈরি গুরুত্বপূর্ণ
▶

VS Code-এ ! টাইপ করে Enter চাপলে নিচের কোড স্বয়ংক্রিয় আসে:

<!-- প্রতিটি HTML ফাইলের মূল কাঠামো -->
<!DOCTYPE html>
<html lang="bn">
<head>
  <meta charset="UTF-8">
  <title>আমার প্রথম পেজ</title>
</head>
<body>
  <h1>স্বাগতম!</h1>
</body>
</html>

Semantic HTMLঅর্থবহ ট্যাগ ব্যবহার
৩-৫ দিন ▼
Semantic ট্যাগ কী এবং কেন? গুরুত্বপূর্ণ
▶

Semantic ট্যাগ ব্রাউজার ও Google-কে বলে দেয় কোন অংশের কী কাজ। SEO-র জন্য অনেক গুরুত্বপূর্ণ।

<header>  <!-- উপরের অংশ, লোগো, মেনু -->
<nav>     <!-- নেভিগেশন মেনু -->
<main>    <!-- মূল কন্টেন্ট -->
<section> <!-- একটি বিভাগ -->
<article> <!-- স্বাধীন লেখা/ব্লগ পোস্ট -->
<aside>   <!-- পাশের তথ্য, বিজ্ঞাপন -->
<footer>  <!-- নিচের অংশ -->
div ও span — সাধারণ কন্টেইনার
▶

div — block level (পুরো লাইন নেয়) | span — inline (লেখার মাঝে ব্যবহার)

<!-- div: CSS দিয়ে গ্রুপ করতে -->
<div class="card">
  <h2>কার্ড শিরোনাম</h2>
  <p>কার্ডের বিষয়বস্তু</p>
</div>

<!-- span: লেখার মাঝে রঙ দিতে -->
<p>এখানে
  <span style="color:red">লাল</span>
  রঙের লেখা।
</p>
সম্পূর্ণ পেজ লেআউট উদাহরণ
▶ 
<body>
  <header>
    <h1>আমার ওয়েবসাইট</h1>
    <nav>
      <a href="#">হোম</a>

    </nav>
  </header>

  <main>
    <section id="about">
      <h2>আমাদের সম্পর্কে</h2>
      <p>আমরা একটি প্রযুক্তি প্রতিষ্ঠান।</p>
    </section>
  </main>

  <footer>
    <p>© 2024 আমার ওয়েবসাইট</p>
  </footer>

৩-৫ দিন ▼
meta ট্যাগ ও SEO গুরুত্বপূর্ণ
▶

meta ট্যাগ Google ও ব্রাউজারকে পেজ সম্পর্কে তথ্য দেয়।

<head>
  <meta charset="UTF-8">

  <!-- মোবাইলে সঠিক দেখানোর জন্য -->
  <meta name="viewport"
        content="width=device-width, initial-scale=1">

  <!-- Google সার্চে বিবরণ -->
  <meta name="description"
        content="আমার ওয়েবসাইটের বিবরণ">

  <title>আমার পেজ | সাইটের নাম</title>
</head>
CSS লিংক করা গুরুত্বপূর্ণ
▶

HTML-এ CSS যোগ করার ৩টি উপায়:

<!-- ১. বাইরের ফাইল (সেরা পদ্ধতি) -->
<link rel="stylesheet" href="style.css">

<!-- ২. head-এর ভেতরে style ব্লক -->
<style>
  body { background: #fff; color: #333; }
</style>

<!-- ৩. সরাসরি ট্যাগে (এড়িয়ে চলুন) -->
<p style="color: red;">লাল লেখা</p>

<!-- body-র একদম শেষে রাখুন -->

<script>
